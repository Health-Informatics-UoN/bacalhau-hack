# Hutch Bunny

We tried running Hutch Bunny (An HDR UK Cohort Discovery Request resolver).

First we tried just asking Bacalhau to run the docker image for this tool directly.

We encountered learning opportunities in the form of needing to pass input files to container workloads in Bacalhau.

## Getting the container image running

Example command for running:

```bash
docker run -it --rm ghcr.io/bacalhau-project/bacalhau:latest docker run --api-host host.docker.internal --wait -e DATASOURCE_DB_USERNAME=postgres -e DATASOURCE_DB_PASSWORD=postgres -e DATASOURCE_DB_HOST=host.docker.internal -e DATASOURCE_DB_SCHEMA=omop -e DATASOURCE_DB_PORT=5432 -e DATASOURCE_DB_DATABASE=omop ghcr.io/health-informatics-uon/hutch/bunny:1.0.0-alpha.4 -- bunny --body
```

This run successfully but fails to connect to the database and produce an output.
